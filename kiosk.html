Publish this project on GitHub and get your own live link

  1. Create a new repository on GitHub and name it something like drishti-form.
  2. Upload all project files into the repository root, including index.html, kiosk.html, and the css and js folders.
  3. Open the repository settings, go to Pages, and choose Deploy from a branch.
  4. Select the main branch and the /root folder, then save.
  5. Wait 1–3 minutes, then open your site at https://your-username.github.io/drishti-form/.
  6. To open the kiosk page directly, visit https://your-username.github.io/drishti-form/kiosk.html.
